1. Information Access

As a user, I want to easily find clear information on the website, so that I can understand the services or activities available.

Acceptance Criteria

The website has clear navigation.

Key information pages are accessible from the main menu.

Content is organised logically and easy to read.

2. Online Booking / Purchase

As a customer, I want to book or purchase services through the website, so that I can plan or secure my place in advance.

Acceptance Criteria

The website includes a booking or purchase system.

Users can select available options before confirming.

Users receive confirmation after completing the booking.

3. User Account

As a returning user, I want to create an account and log in, so that I can manage my bookings or purchases.

Acceptance Criteria

Users can create an account.

Users can log in securely.

Users can view and manage their bookings or purchases.

4. Mobile Accessibility

As a user, I want the website to work well on mobile devices, so that I can access information or services while travelling or on the go.

Acceptance Criteria

The website layout adapts to different screen sizes.

All features are accessible on mobile devices.

Text and buttons are easy to read and interact with.

5. Personalisation / Rewards

As a returning customer, I want to receive rewards or personalised features, so that I feel valued and encouraged to use the service again.

Acceptance Criteria

The system tracks user activity or visits.

Returning users receive rewards, discounts, or personalised content.

Users can view their rewards in their account.
